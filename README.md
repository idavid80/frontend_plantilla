# frontend_plantilla
Construir y desarrollar un de esqueleto frontend con React.

# Instalar React

> npx create-react-app ruta
> cd ruta
> npm start 

# Configurar React

# Jest

> npm install --save-dev jest

> jest --init

{
  "scripts": {
    "test": "jest"
  }
}

## Iniciar Jest

>npm test
